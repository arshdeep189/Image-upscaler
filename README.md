Image Super-Resolution using SRGAN and SwinIRThis repository implements two state-of-the-art approaches for Image Super-Resolution (SR): a Generative Adversarial Network (SRGAN) and a Transformer-based model (SwinIR). The project focuses on upscaling low-resolution (LR) images to high-resolution (HR) versions with $4\times$ magnification while preserving fine structural details.🚀 FeaturesSRGAN Implementation: Utilizes a Generator-Discriminator architecture with VGG-19 perceptual loss to ensure realistic texture reconstruction.SwinIR Fine-tuning: Leverages the Swin Transformer architecture for image restoration, utilizing shifting window self-attention for superior performance over traditional CNNs.Quantitative Evaluation: Includes an evaluation pipeline to measure model accuracy using PSNR (Peak Signal-to-Noise Ratio) and SSIM (Structural Similarity Index).🛠️ Tech StackFrameworks: TensorFlow/Keras (SRGAN), PyTorch (SwinIR).Libraries: HuggingFace Transformers, OpenCV, Scikit-Image, NumPy, Matplotlib.Language: Python.📂 Project Structuremodel.py / Network.py: Contains the Keras/TensorFlow architectures for the Generator and Discriminator.train.py: PyTorch training script for fine-tuning the SwinIR model.evaluate.py: Script to calculate PSNR and SSIM metrics on test datasets.utils.py: Helper functions for image normalization, denormalization, and bicubic downscaling.📈 Results & MetricsThe models are evaluated based on their ability to reconstruct high-frequency details from $24 \times 24$ inputs to $96 \times 96$ outputs.PSNR: Measures the ratio between the maximum possible power of a signal and the power of corrupting noise.SSIM: Measures the perceived change in structural information between the HR and SR images.🏃 How to RunInstall Dependencies:Bashpip install tensorflow torch torchvision transformers scikit-image pillow
Train SwinIR:Bashpython train.py
Evaluate Performance:Bashpython evaluate.py
